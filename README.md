# Heretic Models

[![Hugging Face](https://img.shields.io/badge/🤗%20Hugging%20Face-paoloronco-yellow)](https://huggingface.co/paoloronco)

Uncensored language models (LLMs) created with [Heretic](https://github.com/p-e-w/heretic), an open-source tool that automatically removes safety alignment restrictions from AI models through **abliteration**.

---

## What is Heretic

[Heretic](https://github.com/p-e-w/heretic) is a tool that modifies the weights of a language model to suppress automatic refusals (e.g. "I can't help with that") without retraining the model from scratch. The process is called **abliteration** and runs fully automatically.
